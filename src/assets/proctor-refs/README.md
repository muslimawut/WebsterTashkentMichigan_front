# Allowed-page reference screenshots (AI vision tab-check)

Drop screenshots of the **ALLOWED** exam pages here (PNG/JPG/WEBP). The Proctor
Monitor loads every image in this folder automatically as a reference — no upload
needed in the UI.

Recommended:
- `metrica-grammar.png` — Cambridge Metrica grammar question layout
- `metrica-listening.png` — Cambridge Metrica listening/audio layout
- `metrica-reading.png` — Cambridge Metrica reading passage layout
- `mept.png` — the MEPT proctoring page

The AI vision "Check tabs" compares the student's screen snapshots against these
references; any snapshot showing a different tab/site is flagged as cheating.

Notes:
- Any number of images works; the code globs `*.png|jpg|jpeg|webp` in this folder.
- Keep them reasonably sized (they're sent to the model as images).
